# Morva Koffi

Static GitHub Pages site for Morva Koffi, a takeaway seaside coffee shop near Barnoon Long Stay car park in St Ives.

## Local Preview

Open `index.html` directly, or run a simple static server:

```sh
python3 -m http.server 8080
```
